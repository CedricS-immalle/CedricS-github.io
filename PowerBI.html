Power BI

Power BI is een tool om data te visualiseren. Het is een zeer slim programma omdat het zelf gegevens kan interpreteren en er de correcte grafieken bij ontwerpt. Het programma heeft een versie in de cloud gekend als Power BI Services maar ook een desktop interface genaamd Power BI desktop.

In Power BI kan je van alle soorten data importeren. Bijvoorbeeld Google Analytics, Excel documenten, Github repository, etc. In twee korte instructievideo's tonen we hoe dit gaat voor Excel en Github
